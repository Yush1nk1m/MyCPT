## 점수화 방식

```
Most 선택  → 해당 척도 +2점
Least 선택 → 해당 척도 -1점
미선택     → 0점

24문항 기준 각 척도 원점수 범위: -24 ~ +48
→ 정규화해서 1~9 버킷으로 변환
```

## 결과 해석 시 유의점

이 검사는 **강제선택(자가준거형)** 방식입니다. 즉 결과는 "내 안에서 D·I·S·C 중 무엇이 더 강한가"라는 **개인 내부의 상대적 순위**를 보여줍니다. 따라서 점수의 절댓값을 다른 사람과 직접 비교하기보다는, **자신의 가장 두드러진 성향과 보조 성향**을 읽는 용도로 해석하는 것이 적절합니다.

## 개정 노트

원본 24문항을 유지하되, 신뢰도와 공정성을 해치는 부분을 다음 기준으로 다듬었습니다.

1. **평가적 중립화** — 네 선택지가 비슷한 수준으로 "괜찮은 방식"이 되도록 조정했습니다. 특정 유형을 고르면 손해 보는 느낌이 드는 불균형을 없앴습니다.
2. **결함 연상 표현 제거** — S 선택지에 반복되던 수동·우유부단 묘사("그냥 넘어간다", "아무 데나 상관없으니", "불만 있어도 따른다")와 D의 독단 묘사를, 각 유형의 진짜 동기(S=조화·꾸준함, D=목표추진)를 살린 긍정적 표현으로 바꿨습니다.
3. **선택지 길이 균형** — 한 문항 안에서 선택지 길이를 비슷하게 맞춰, 길이 차이로 인한 시선 쏠림을 줄였습니다.
4. **일상성 유지** — 가이드라인의 격식체 비즈니스 서술·직무 도메인 매핑은 적용하지 않았습니다. "누구나 쉽게 측정"이라는 목표에 맞춰 친숙한 일상 상황과 말투를 그대로 살렸습니다.
