## 점수화 방식

```
Most 선택  → 해당 척도 +2점
Least 선택 → 해당 척도 -1점
미선택     → 0점

24문항 기준 각 척도 원점수 범위: -24 ~ +48
→ 정규화해서 1~10 버킷으로 변환
```
